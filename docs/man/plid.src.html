Name

plid - the Pipeline hardware ID tool

Synopsis

plid [options]

plid --help
plid --html-help
plid --version
plid --release-date
plid --copyright

Description

This program is used to generate a unique hardware ID for the hosts which will run one of the Pipeline server daemons (see the --master-host, --queue-host and --file-host options to plconfig(1)).

When plid(1) is run, it outputs a lines composed of the fully resolved hostname of the local machine and the hardware ID for that host.  The output for each of the server hosts should be placed in a plain text file with one hostname per line (see --append).  The path to this host  IDs file is then given as the argument to the --host-ids option to plconfig(1) when generating the site configuration.

Information Options

The following options provide information about plid(1) and then immediately exit:
--help
Prints a brief usage message and exits.
--html-help
Opens this page using the mozilla(1) web browser. 
--version
Prints the version number of this software and exits.
--release-date
Prints date that this software was released and exits.
--copyright

Prints copyright notice for this software and exits.

--license

Prints license agreement for this software and exits.

Options

--append=filename

Specifies that the host ID information should be appended to the given file (filename) instead of printed to stdout.  If the file is located on a network mounted filesystem, this option can be used to more easily generate the file passed to the --host-ids option of plconfig(1) by simple running the same command line on each host.
